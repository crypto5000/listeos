Comments

Comments are starting responses to the topic. To comment, you send an EOS transaction from a wallet. The transaction must: 1) be sent to the Topic ID address; 2) contain the specified EOS amount; 3) include a text memo. The "memo" field is the text that gets displayed on Listeos as your comment. The account you send "from" becomes your username displayed next to the comment.

Replies

You reply to starting comments or other replies. Replies take the same form as comments EXCEPT the memo field requires a special starting code. The memo field must start with an id, followed by a colon ":". An example memo, would be "90c17b59: This is my reply". In this example, the text "This is my reply" would be displayed.

Upvote #1

Upvotes push comments or replies higher. To upvote, you send an EOS transaction from a wallet. The transaction must: 1) be sent to the Topic ID address; 2) contain the specified EOS amount; 3) include a specially formatted memo. The memo field must start with an id, followed by the text ":up". An example memo, would be "90c17b59:up".

Upvote #2

Upvotes push comments or replies higher. To upvote, you send an EOS transaction from a wallet. The transaction must: 1) be sent to the user's address being upvoted; 2) contain the specified EOS amount; 3) include a specially formatted memo. The memo field must start with the topic ID, followed by the text ":up:", followed by a unique id. An example memo, would be "aaaaaaaaaaaa:up:90c17b59".

Downvotes

Downvotes push comments or replies lower. To downvote, you send an EOS transaction from a wallet. The transaction must: 1) be sent to the Topic ID address; 2) contain the specified EOS amount; 3) include a specially formatted memo. The memo field must start with an id, followed by the text ":down". An example memo, would be "90c17b59:down".

Wallets

Listeos works with EOS transactions. So any wallet or EOS tool that allows you to transfer EOS should work with Listeos. You just need to be able to type the appropriate text into the "memo" field of the transaction. It's also possible to send from an exchange - assuming the exchange doesn't have a minimum withdrawal limit. If you send from an exchange, your displayed username is the exchange's wallet address.

Frequently Asked Questions:

Question: What is Listeos?

Answer: Listeos is a discussion platform for the EOS community. It uses the EOS blockchain to comment, vote, and reward people.

Question: Why use Listeos over existing alternatives?

Answer: Listeos uses EOS microtransactions to reward people and deter spam. Plus, it's decentralized and censorship resistant.

Question: How fast will a comment show on Listeos?

Answer: All transactions (including comments and upvotes) are processed in under 1 second on the EOS blockchain.

Question: Do I need an EOS account to use Listeos?

Answer: You need EOS to use Listeos. It's best to have your own account, but it's still possible to use Listeos from an exchange (without having your own account).

Question: What are Topic IDs?

Answer: Topic IDs are just EOS accounts from the genesis snapshot. Every topic is linked to 1 EOS account. So every discussion has the potential to send small amounts of EOS to a specific tokenholder.

Question: How are genesis accounts allocated to topics?

Answer: Each topic randomly pulls a genesis account that hasn't been allocated to a discussion yet. Once all genesis accounts have been used, the cycle starts over.

Question: Why do I have a bunch of small EOS transfers in my account?

Answer: EOS transactions within Listeos go from tokenholder to tokenholder. It's sort of like an airdrop from another person directly to you.

Question: Does it cost EOS to use Listeos?

Answer: All transactions require tiny amounts of EOS. This helps deter spam comments and provide rewards to users.

Question: How much EOS does it cost to make a comment?

Answer: These are microamounts like 0.000123 EOS.

Question: Do I have to send the exact EOS specified by the topic's instructions?

Answer: Yes. This is to minimize collisions. If you send a different amount, Listeos ignores it.

Question: Can I make EOS using Listeos?

Answer: Listeos allows for upvotes directly to users. So if someone likes your response enough to upvote, they may send you EOS directly.

Question: How do I comment, upvote, downvote, and reply?

Answer: See the "How To" section.

Question: What's the difference between a comment and reply?

Answer: Comments are top-level starting responses to the topic. Everything else is a reply. You reply to comments. Then you reply to replies.

Question: Is there a limit to a comment or reply?

Answer: The limit is 256 characters, which is the maximum you can put into the "memo" field of a transfer.

Question: What wallets are supported?

Answer: Any wallet that allows you to transfer EOS and has a memo field.

Question: Does Listeos ask for my private keys?

Answer: No.

Question: Why are there 2 options for upvoting?

Answer: This is to deter sybil attacks, where a person upvotes themselves.

Question: If someone upvotes me, do I get EOS?

Answer: Only if they choose to send you EOS directly, rather than sending to the genesis-topic account.

Question: What is that unique id used in the memo?

Answer: This is to identify a specific comment or reply. A person could make multiple replies within a single topic. The id is derived from the blockchain transaction ID.

Question: Can I use Listeos from an exchange?

Answer: Yes, assuming the exchange has a transfer feature with a memo field and allows small withdrawals (< 0.1EOS). Note that the exchange account becomes your user name for the comment.

Question: Is the code open source?

Answer: Yes. It's on github here.

Question: What coins are supported?

Answer: Currently, just EOS.
